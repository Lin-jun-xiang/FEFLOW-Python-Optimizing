# Water Retention Curve

### 簡介

* 繪製 "**未飽和曲線**"

* 輸入參數:
    * `particle_types`
        > 岩層屬性，例如礫石層、砂層或泥層
    * `saturation_capacity` [無因次/無量綱]
        > 飽和含水量，一般設為 1
    * `residual_saturation` [無因次/無量綱]
        > 殘餘含水量，定義為水分含量的變化梯度 (dθ /dh) 為零的時候，此時的水分含量即為殘餘水分含量
    * `air_entry_value` [1/m]
        > 與空氣陷入值有關，a 越大曲線越陡
    * `pore_size_distribution` [無因次/無量綱]
        > 與土壤孔徑分布有關，土壤孔徑分布越均勻n越小，曲線越圓滑、平緩，n值越大

---
